telegramBotPhp

مشاهده در گیتهاب
برگشت به صفحه اصلی

منظور از آپدیت ها متد هایی هستند که موقع ارسال پیام توسط کاربر قابل استفاده هستند

این متد ها دارای زیرمجموعه ای از متدها هستند که به صورت متد های زنجیره ای میتوان از آنها استفاده کرد که با استفاده از get{name} گرفته می شوند و توسط isset{name} اعتبار سنجی می شوند

اگر میخواهید اطلاعات بیشتری از متد های زنجیره ای به دست آورید اینجا کلیک کنید

متد message :

داده هایی از نوع message هستند که کاربر به صورت عادی برای ما ارسال می کند

مقدار برگشتی : Message
مستند تلگرام : مشاهده

نمونه کد :

$Telegram->message();

متد edited_message :

این متد هنگامی که یک پیام ویرایش شود قابل استفاده است

مقدار برگشتی : Message
مستند تلگرام : مشاهده

نمونه کد :
