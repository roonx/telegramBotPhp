--- title: نصب و راه اندازی description: یک بیس برای ساخت ربات تلگرام ---

جهت نصب و راه اندازی بیس ابتدا دستور زیر را در cmd اجرا کنید

composer require roonx/telegram-bot-api-php

‫پس از نصب با استفاده از include بیس رو فراخوانی کنید : ‬

<?php include 'vendor/autoload.php'; ?>

پس از فراخوانی یک شی بسازید و از بیس استفاده کنید

<?php include 'vendor/autoload.php'; $Telegram = new telegramBotApiPhp\telegram('token'); ?>
